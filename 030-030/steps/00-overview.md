# Lab 030-030: KMCP Controller Installation

## Overview

In this lab, we'll explore the KMCP (Kubernetes MCP) controller installation and its functionality. The KMCP controller is essential for managing MCP servers in a Kubernetes environment.

## Lab Steps

1. **Step 1: Basic Kagent Setup**
   - Configure a minimal Kagent installation
   - Verify basic functionality
   - Prepare the environment for KMCP

2. **Step 2: KMCP Controller Installation**
   - Install KMCP CRDs using Helm
   - Validate the installation
   - Understand the components

## Prerequisites

- Kubernetes cluster access
- kubectl command-line tool installed
- Helm v3.x installed
- Basic understanding of Kubernetes concepts
- Basic understanding of Custom Resource Definitions (CRDs)

## Expected Duration

- Approximately 20-30 minutes

## Learning Objectives

By completing this lab, you will:
- Understand KMCP controller's role in the Kagent ecosystem
- Learn how to install and validate KMCP components
- Gain experience with Kubernetes CRDs and controllers
- Prepare your cluster for MCP server deployments

# Lab 030-090: MCP Server Deployment and Development

## Overview

This lab focuses on two main objectives:

1. Deploying an existing MCP server (AWS Pricing MCP Server) using KMCP
2. Developing and deploying a local MCP server with AI agent integration

## Lab Steps

1. **Step 1: Deploy AWS Pricing MCP Server**
   - Configure and deploy the AWS Pricing MCP server using KMCP
   - Learn about MCP server configuration and deployment patterns
   - Note: We'll use a simplified IAM setup for this demonstration

## Prerequisites

- Kubernetes cluster access
- Basic understanding of MCP (Managed Control Plane) concepts
- kubectl command-line tool installed
- kmcp CLI tool installed (`npm install -g kmcp`)
- Docker for building and pushing images
- Python 3.8+ for local development

## Expected Duration

- Approximately 30-45 minutes

## Learning Objectives

By completing this lab, you will:
- Understand MCP server deployment using KMCP
- Learn about MCP server configuration and integration
- Gain hands-on experience with Kubernetes-based MCP deployments
